Inline [Lucide](https://lucide.dev) icon — the brand's specified outline icon system. Load the Lucide UMD script on the page first: `<script src="https://unpkg.com/lucide@latest"></script>`.

```jsx
<Icon name="shopping-cart" size={20} />
<Icon name="wrench" size={18} color="var(--orange-600)" />
```

Accepts kebab-case or PascalCase names. Renders inline SVG (survives re-renders). Use clean outline icons only — no colorful icon packs.
