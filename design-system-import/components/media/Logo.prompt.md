Renders the real Cellular Solutions brand logo from `assets/`. Use `tone="dark"` on dark (graphite) backgrounds and `variant="icon"` for the square mark alone.

```jsx
<Logo height={44} />
<Logo variant="icon" tone="dark" height={40} />
```

Set `basePath` if your page isn't at the project root (e.g. `basePath="../../assets"`).
